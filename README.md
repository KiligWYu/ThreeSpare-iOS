## Description

[每日一文](http://meiriyiwen.com) 非官方客户端。

『冬者岁之余，夜者日之余，阴雨者时之余也。』是谓三余。

> 每天花 10 分钟阅读一篇文章，一个月可以有大约 50000 字的阅读量，一年有近 60 万字的阅读量，专注、执着，每天阅读。
> 
> 只为简单的纯净的阅读而生。
> 
> 简单生活，每日一文。
